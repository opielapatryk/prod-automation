Main API Endpoints

Routes
  • GET /api/collector/routes/list/ - List all routes
  • GET /api/collector/routes/{id}/ - Get route details
  • POST /api/collector/routes/optimize/ - Optimize a route
  • POST /api/collector/routes/create/ - Create a route
Machines
  • GET /api/collector/machines/ - Get all machines
  • GET /api/collector/machines/warnings/ - Machines with warnings
Telemetry
  • POST /api/collector/telemetry/receive/ - Submit telemetry data
